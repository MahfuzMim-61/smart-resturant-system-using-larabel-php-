# Luxe Dining - Kivabe Run Korben?

Project ti chalu korar jonno nicher dhap gulo onushoron korun:

### 1. Database Chalu Korun
*   **XAMPP Control Panel** open korun.
*   **MySQL** er `Start` button e click korun. (Apache start na korleo cholbe, karon amra artisan serve use korchi).

### 2. Project Folder e Jan
Apnar computer er **Terminal** ba **Command Prompt (CMD)** open korun ebong nicher command likhun:
```bash
cd "e:\luxe\luxe-app"
```

### 3. Server Start Korun
Project folder e dhakar por ei command ti din:
```bash
C:\xampp\php\php.exe artisan serve
```
Ykhon apni dekhben: `Server running on [http://127.0.0.1:8000]`

### 4. Browser e Jan
Google Chrome ba onno browser open kore ei thikanay jan:
[http://localhost:8000](http://localhost:8000)

---
**Bondho Korar Niyom:**
Server bondho korte chaile Terminal e giye `Ctrl + C` chapun.
